# Flutter - News App

A Simple News App built with Flutter. I used API From - [NewsAPI](https://newsapi.org/), Bloc State Management and Dio.

# Features
- Easy to use
- Search News
- Dark mode & Light mode
- Interactive UI
- Sort By Channel
